# PhotoBook Studio - 产品规格书

## 概述
PhotoBook Studio 是一款面向 Windows 用户的画册/相册排版桌面工具，支持图片排版、文字编辑、页面管理、PDF/图片导出。

## 技术栈
- **Electron** (桌面应用框架)
- **Fabric.js** (画布引擎，CDN引入)
- **jsPDF + html2canvas** (导出PDF和图片，CDN引入)
- **esbuild** (轻量打包工具)
- **Node.js**

## 页面尺寸
- A5: 148 × 210 mm (416 × 595 px @ 72dpi)
- A4: 210 × 297 mm (595 × 842 px @ 72dpi)
- 自定义尺寸

## 功能列表

### 1. 图片管理
- 从文件夹批量导入图片
- 拖拽文件到应用窗口导入
- 左侧缩略图面板显示所有已导入图片
- 点击缩略图选中，拖入画布添加

### 2. 画布编辑器
- 基于 Fabric.js 的交互式画布
- 选中对象：显示控制框（缩放/旋转/删除）
- 支持多选（Shift+点击）
- 键盘快捷键：Delete 删除选中，Ctrl+Z 撤销
- 画布缩放：10%-400%
- 滚动条/平移支持

### 3. 文字工具
- 点击添加文字框
- 属性面板控制：字体（系统字体列表）、字号（8-144pt）、颜色、加粗、斜体、对齐

### 4. 页面管理
- 左侧页面缩略图面板
- 添加新页面（空白）
- 删除当前页面
- 页面排序（拖拽）
- 当前页面高亮

### 5. 预设模板
- 1张图：全页填充（留白边）
- 2张图：上下分布
- 3张图：1大2小
- 4张图：2×2网格
- 网格模板：3×3

### 6. 图片属性（右侧面板）
- 边框：颜色+宽度
- 阴影：开关+偏移+模糊+颜色
- 圆角
- 不透明度
- 旋转角度

### 7. 导出功能
- 导出为 PDF（jsPDF，多页）
- 导出当前页为 PNG
- 导出全部页面为 PNG（逐页）

### 8. 菜单
- 文件：导入图片、导出PDF、导出PNG、退出
- 编辑：撤销、重做、删除、缩放
- 帮助：关于

- 文字：#e2e8f0

### 布局
```
┌─────────────────────────────────────────────────────┐
│  菜单栏 / 工具栏 (48px)                              │
├──────────┬─────────────────────────┬─────────────────┤
│ 页面列表  │                         │   属性面板      │
│ (160px)  │      画布区域            │   (220px)      │
│          │   (可缩放/平移)          │                │
│ 缩略图    │                         │ 图片属性        │
│ 列表     │                         │ 文字属性        │
│          │                         │ 页面设置        │
└──────────┴─────────────────────────┴─────────────────┘
```

### 工具栏
- 选择工具 (V)
- 文字工具 (T)
- 添加页面 (+)
- 删除页面 (-)
- 撤销 (Ctrl+Z)
- 重做 (Ctrl+Y)
- 缩放下拉
- 模板下拉
- 导出按钮
